# 🎮 Tic-Tac-Toe (Java Swing)

A simple and interactive **Tic-Tac-Toe** game built using **Java Swing**. This project demonstrates GUI development, event handling, and game logic in Java.

---

## ✨ Features

* 🎲 Two-player gameplay (Player X vs Player O)
* 🖥️ Interactive graphical user interface using Java Swing
* 🏆 Automatic winner detection
* 🤝 Draw detection
* 🎨 Winning tiles are highlighted
* ⚡ Simple, responsive, and easy-to-understand code

---

## 🛠️ Technologies Used

* Java
* Java Swing
* AWT

## 🚀 Getting Started

### Prerequisites

* Java JDK 17 or above
* IntelliJ IDEA / Eclipse / VS Code

### Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Hitesh-Dey/Tic-Tac-Toe.git
```

2. Open the project in your preferred Java IDE.

3. Compile and run:

```bash
javac TIC_TAC_TOE.java
java TIC_TAC_TOE
```

Or simply run the `main()` method from your IDE.

---

## 🎮 Gameplay

* Player **X** always starts first.
* Players take turns placing their mark.
* The first player to align **3 symbols** horizontally, vertically, or diagonally wins.
* If all cells are filled without a winner, the game ends in a **Draw**.
